Variable

Variable is just like a container used to store a value.

Javascript is a dynamic language means that we do not have to define a variable data type when we declare the variable.

a=20
age = 27
price = 28.99
Name = "Faizan"

Rules to define a Variable

  1. Javascript is a case sensitive language so "A" & "a" both are diffrent
  2. Variable name containes $, underscore (_), letters[(A-Z), (a-z)], digits except space
  3. Only a letter[(A-Z), (a-z)], underscore(_) or $ should be a 1st character
  4. Reserved word cannot be a variable name like console, log, function, var, let, etc...

Let, Var, Const

Let

Variabel cannot be re-declare but can be updated.

A block scope variable

Var

Variable can be re-declare and updated

A global scope variable

Const

Variable cannot be re-declare or updated.
